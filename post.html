Real Time Chat Application

Developed a real-time chat application using Django, Django Channels, and HTMX, enabling dynamic WebSocket-based messaging with features like public chat rooms, group chats, and private conversations.

Integrated Django Allauth for secure and customizable user authentication, supporting seamless login, registration, and social authentication.

Implemented an online user tracking system to display active users in real-time, enhancing the interactive experience for chat participants.

Designed a scalable architecture to manage concurrent users and message persistence, ensuring reliability and responsiveness in real-time communication.

Technologies:
  • - Python
  • - Django
  • - HTMX
  • - Postgres
  • - Redis
  • - Django Channels
  • - Django-allauth
  • - CSS
